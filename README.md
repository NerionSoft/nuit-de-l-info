# LearnLinux

Simulateur Linux dans le navigateur - Projet Nuit de l'Info 2025

## C'est quoi ?

Un simulateur Linux complet qui tourne direct dans ton navigateur. Pas besoin d'installer quoi que ce soit, tu lances et tu apprends.

- Terminal avec les vraies commandes (`ls`, `cd`, `cat`, `mkdir`, etc.)
- Gestionnaire de fichiers style GNOME
- Editeur de texte
- Tutoriel guide pour les debutants

## Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS
- Framer Motion

## Lancer le projet

```bash
pnpm install
pnpm dev
```

Ouvre [http://localhost:3000](http://localhost:3000)
